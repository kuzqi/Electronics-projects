A simple automatic night light with LDR contains 3 stages of light level checker made with arduino

## Components 
- Arduino UNO
- Jumpers Wires
- Button
- LDR (Photoresistor)
- 2x 100 ohm resistors
- 2x 200 ohm resistors
- LEDs
